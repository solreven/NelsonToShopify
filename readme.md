## Making CSV-files Shopify-friendly

### This is a simple Typescript program to parse Nelson Garden CSV-files for article numbers and transforming them into Shopify-compatible SKU-codes. Written using jest for testing.

### SETUP

- Get Node
- Get Papaparse
- Run
